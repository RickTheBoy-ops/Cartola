## 🚀 Quick Start (RECOMENDADO)

### **Modo Principal - ULTIMATE OPTIMIZER (AI-Powered)**

```bash
cd src
py cartola_ultimate_optimizer.py
```

**O que ele faz:**
1. 📡 Busca dados do Cartola FC (3 APIs oficiais)
2. 📡 Consulta RapidAPI para forma dos times (WWWWW = +15% boost)
3. 🧮 Otimização matemática (sem conflitos de adversários)
4. 🤖 Validação automática com Perplexity IA
5. 📊 Display unificado (Tabela + Análise IA)
6. 💾 Salva Excel com 2 abas (Escalação + Relatório IA)

---

## 🎯 Diferenciais

### ✅ **Inteligência Artificial Integrada**
- **Perplexity IA** valida TODA escalação automaticamente
- Verifica notícias de última hora (lesões, poupados)
- Analisa matchups
- Dá nota de segurança (0-10)
- Sugere substituições se necessário

### ✅ **Dados em Tempo Real**
- **RapidAPI** busca forma dos times (últimos 5 jogos)
- Multipliers automáticos: WWWWW = 1.15x, WWW = 1.10x
- Aplicados automaticamente nos scores

### ✅ **Regras de Profissional**
- **ZERO conflitos** de adversários diretos
- Nunca escala defensor de um time + atacante do adversário
- Filtra apenas jogadores DE ELITE com histórico
- Estratégias POINTS ≠ WEALTH claramente diferenciadas

### ✅ **Otimização Inteligente**
- Usa TODO o orçamento disponível
- Evita jogadores sem histórico (média = 0)
- Prioriza os MELHORES jogadores
- Formações: 4-3-3, 3-4-3, 3-5-2, 4-4-2

## � Configuração de Chaves

- Defina as chaves via variáveis de ambiente (arquivo `.env`):
  - `RAPIDAPI_KEY`
  - `PERPLEXITY_API_KEY`
- Crie um arquivo `.env` na raiz do projeto (não comitar) com:
  ```
  RAPIDAPI_KEY=seu_token_aqui
  PERPLEXITY_API_KEY=seu_token_aqui
  ```
- Consulte `.env.example` para ver todas as variáveis necessárias.

**⚠️ IMPORTANTE:** Nunca comite chaves reais. Use `.env` local e versionar apenas `.env.example`.

## 🔧 Troubleshooting

### **Problema: APIs não respondem**
```
⚠️ RapidAPI não retornou dados. Usando multipliers padrão.
⚠️ IA indisponível. Análise baseada apenas em estatísticas.
```
**Solução:** O sistema continua funcionando com fallback automático.

### **Problema: Poucos jogadores elegíveis**
```
❌ Só conseguiu 8 jogadores sem conflitos
```
**Solução:** Aumente o orçamento ou escolha estratégia WEALTH (menos restritiva).
